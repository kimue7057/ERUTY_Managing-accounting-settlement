# ERUTY 향후 개선 기능 백로그

이 문서는 현재 보류 중인 기능과 향후 개선 후보를 정리한 백로그입니다.

- 상태 기준
  - `Backlog`: 아직 구현하지 않았고 추후 검토 예정
  - `Planned`: 구현 우선순위가 높아 구체화가 필요한 항목
  - `Later`: 중장기 검토 항목

## 1. 계정 / 인증

### Backlog

- 회원가입 신청 기능
  - 일반 사용자가 직접 계정 사용을 요청할 수 있는 신청 폼 및 대기 상태 관리
- 관리자 가입 승인
  - 관리자 또는 운영자가 신규 가입 신청을 검토하고 승인 / 반려 처리
- 비밀번호 재설정 UX 개선
  - 비밀번호 재설정 메일, 안내 문구, 완료 흐름, 만료 처리 UX 보완

## 2. 권한 / 보안

### Planned

- 권한별 화면 제한 고도화
  - 단순 페이지 접근 제어를 넘어 버튼, 액션, 데이터 범위 단위까지 세분화
- RLS 실제 운영 정책 강화
  - 개발 편의 위주의 예외를 줄이고 실제 운영 권한 기준으로 정책 세분화
- 감사 로그
  - 주요 데이터 변경 이력과 민감 액션 이력 저장
- 관리자 활동 기록
  - 승인, 반려, 설정 변경, 자금 수정, 정산 확정 등 관리자 행동 추적

## 3. 알림 / 커뮤니케이션

### Backlog

- 이메일 알림
  - 승인 요청, 반려, 수정요청, 정산 확정, 지급 완료 등 이벤트 알림
- 슬랙 / 카카오 알림
  - 내부 협업용 실시간 알림 채널 연동

## 4. 다운로드 / 문서화

### Backlog

- 엑셀 xlsx 다운로드
  - CSV를 넘어 다중 시트, 서식, 요약표가 포함된 실제 회계 전달용 파일 생성

## 5. 자동화 / 외부 연동

### Later

- 영수증 OCR
  - 업로드한 영수증에서 사용처, 금액, 날짜 자동 추출
- 은행 API 연동
  - 실제 계좌 잔액 및 거래 내역 자동 동기화
- 법인카드 내역 자동 연동
  - 카드 승인 내역과 지출 요청 간 자동 매칭 또는 대조

## 6. UI / UX

### Planned

- 모바일 UX 개선
  - 테이블 밀도 조정, 카드형 전환, 필터 사용성, 업로드 흐름 최적화

## 권장 구현 순서

1. 회원가입 신청 기능
2. 관리자 가입 승인
3. 권한별 화면 제한 고도화
4. RLS 실제 운영 정책 강화
5. 비밀번호 재설정 UX 개선
6. 이메일 알림
7. 관리자 활동 기록
8. 감사 로그
9. 엑셀 xlsx 다운로드
10. 슬랙 / 카카오 알림
11. 모바일 UX 개선
12. 영수증 OCR
13. 법인카드 내역 자동 연동
14. 은행 API 연동

## 메모

- 위 항목들은 현재 문서화만 되어 있으며, 실제 구현은 포함하지 않습니다.
- 운영 환경 전환 전에는 권한 정책, 감사 로그, 관리자 활동 기록을 우선 검토하는 것을 권장합니다.
